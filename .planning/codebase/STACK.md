# Stack & Dependencies

## Core Technologies
- **Framework**: React 18
- **Build Tool**: Vite
- **Styling**: Vanilla CSS (CSS Variables for Theming)

## Key Libraries
- `lucide-react`: Para iconografia (usado extensivamente na barra lateral e nos cabeçalhos dos guias).
- `react-dom`: Renderização.

## System Dependencies
- Node.js (v24.18.0)
- Npm (Package Manager)

## Project Context
Web App interativa operando como a Enciclopédia definitiva e Guia de Progressão ("Warlord's Path") para o modpack de Minecraft "Astron City".
